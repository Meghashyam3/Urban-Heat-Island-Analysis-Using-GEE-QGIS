# Urban Heat Island Analysis Using GEE & QGIS

This project focuses on analyzing the **Urban Heat Island (UHI)** effect using satellite imagery and geospatial tools. It was developed as part of the **Indian Space Academy Summer Internship Programme 2026**.

The analysis was carried out using **Google Earth Engine (GEE)** for satellite data processing and **QGIS** for visualization and map preparation.

## Project Objective

The main objective of this project is to study the Urban Heat Island effect by analyzing:

- Land Surface Temperature (LST)
- Normalized Difference Vegetation Index (NDVI)
- Normalized Difference Built-up Index (NDBI)

The project also examines how vegetation and built-up areas influence surface temperature.

---

## Technologies Used

- Google Earth Engine (GEE)
- QGIS
- Landsat 8 Satellite Data
- JavaScript (GEE)

---

## Project Workflow

1. Prepare the Area of Interest (AOI)
2. Process Landsat 8 satellite imagery
3. Generate LST, NDVI, and NDBI layers
4. Export raster outputs from GEE
5. Create thematic maps in QGIS
6. Analyze the relationship between temperature, vegetation, and built-up areas
